### 6.2 1球処理フロー

```
投手が球種選択
  → 投球実行（制球誤差をガウス分布で計算）
    → ゾーン判定（ストライク/ボール）
      → 打者判断（振る or 見る）
        → [振った場合] スイング結果
          → 空振り / ファウル / フェア打球
            → [フェア打球] 打球方向・種類・飛距離計算
              → 守備処理 → アウト or ヒット判定
        → [見た場合] ストライク / ボール加算
```
