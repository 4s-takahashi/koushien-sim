## 基本操作

- [ ] ゲームを開始できる（学校名・都道府県・監督名を入力してスタート）
- [ ] ホーム画面が表示される（練習メニュー・チーム概要が見える）
- [ ] 練習メニューを選択できる（7種類から選べる）
- [ ] 「練習して1日進む」ボタンが動作する
- [ ] 「1週間まとめて進む」ボタンが動作する
- [ ] チーム一覧が表示される（`/team` ページ）
- [ ] 選手一覧に背番号・ポジション・総合力が表示される
- [ ] 選手詳細ページが表示される（`/team/:id`）
- [ ] 選手の能力値バーグラフが表示される
- [ ] OB画面が表示される（`/ob` ページ）
- [ ] ニュースが表示される（ホーム画面下部）
- [ ] 複数日連続で進めることができる（エラーが出ない）

## スカウト

- [ ] スカウト画面が表示される（`/scout` ページ）
- [ ] 中学生一覧が表示される
- [ ] フィルター（学年・ポジション）が動作する
- [ ] 視察ができる（スカウトレポートが表示される）

## セーブ/ロード

- [ ] セーブ画面を開くことができる（ホーム画面のセーブボタン）
- [ ] 手動スロット1にセーブできる
- [ ] 手動スロット2にセーブできる
- [ ] 手動スロット3にセーブできる
- [ ] 手動スロットからロードできる
- [ ] ロード後にゲームを再開できる（練習・日進行が動作する）
- [ ] 自動セーブ（AUTO_MONTHLY）が表示される（1ヶ月以上進行後）
- [ ] 年度替わり後にロードできる
- [ ] ブラウザリロード後もセーブデータが残る
- [ ] ストレージ使用量が表示される

## 年度替わり（長時間プレイ）

※ 1年分（約365日）進行が必要です。

- [ ] 3月31日から4月1日に進む際にエラーが出ない
- [ ] 年度替わり後に3年生が卒業している（選手数が減る）
- [ ] 新1年生が入学している（選手数が補充される）
- [ ] OB画面に卒業生が追加されている
- [ ] 年度替わり自動セーブが作成されている
